Lekce 1: Proměnné

V proměnné se udržuje hodnota, se kterou algoritmus pracuje. Tato hodnota je ale pro různé případy jiná - proměnná. Pro jednoduchost si proměnné představ jako krabičky do kterých se ukládají informace. Java a C# jsou tzv. typové jazyky, což znamená, že o každé proměnné musíš řící, jakého je typu. Nejpoužívanější primitivní datové typy jsou:

  1. Boolean

    2. Nabývá hodnot 'true' a 'false'. Značí, zda něco je, či není.

  2. Integer (int)

    3. Celé číslo

  3. Float, double, long

    4. Desetinná čísla, nejčastěji používán double

  4. Character (char)

    5. Právě jeden znak (např. písmeno, číslice, bílí znak,...)

  5. String

    6. Textový řetězec, libovolné znaky v libovolném počtu. POZOR: v Javě string není primitivní datový typ.

Zůstaneme-li u přirovnání ke krabičkám, v podstatě říkáš, jaký druh krabičky se má použít, jak velká krabička. Jinou krabičku potřebuješ na uložení telefonního čísla a jinou na text dopisu.
Pokud chceš s proměnnou pracovat, je třeba ji neprve založit - inicializovat. Pak jí můžeš přiřadit hodnotu, měnit hodnotu nebo jinak s ní pracovat. Inicializace vypadá následovně: 'typ jménoPromenne;' nebo 'typ jmenoPromenne = vychoziHodnota;'. Všechny primitivní datové typy mají výchozí hodnotu automaticky.

//inicializace promennych
boolean anoNe = true;
int cislo1 = 33;
float cislo2= f1.0;
double cislo3 = -45.11;
long cislo4 = 54632300000000000000000000000000000000;
char znak = 'a'; // je nutne pouzit jednoduche uvozovky
// C# je nutne pouzit slozene uvozovky
string text = "zcela libovlny text";
// Java je nutne pouzit slozene uvozovky
String text = "zcela libovlny text";

// prace s promennou
int vekManzela = 30;
int vekZeny = 28;
int vekovyRozdil = vekManzela - vekZeny;

Úloha

Jakě typy jsou nejvýhodnější pro uložení telefonního čísla, ceny, emailu, poznávací značky, pohlaví?
Napiš algoritmus, který prohodí hodnoty dvou proměnných 'a' a 'b' libovolného typu.
