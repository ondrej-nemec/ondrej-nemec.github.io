Lekce 1: Proměnné

V proměnné se udržuje hodnota, se kterou algoritmus pracuje. Tato hodnota je ale pro různé případy jiná - proměnná. Pro jednoduchost si proměnné představ jako krabičky do kterých se ukládají informace. Java a C# jsou tzv. typové jazyky, což znamená, že o každé proměnné musíš říci, jakého je typu. Při výuce algoritmů budeš nejčastěji používat primitivní datové typy, přičemž nejpoužívanější jsou:

  1. Boolean

    2. Nabývá hodnot 'true' a 'false'. Značí, zda něco je, či není.

  2. Integer (int)

    3. Celé číslo

  3. Float, double, long

    4. Desetinná čísla, nejčastěji používán double

  4. Character (char)

    5. Právě jeden znak (např. písmeno, číslice, bílý znak,...)

  5. String

    6. Textový řetězec, libovolné znaky v libovolném počtu. POZOR: v Javě string není primitivní datový typ.

Na všechna čísla lze použít základní matematické operace (+, -, /, *, modulo). Pomocí '+' se skládají stringy do nových stringů. Skládáním charů vznikají stringy.

Zůstaneme-li u přirovnání ke krabičkám, v podstatě říkáš, jaký druh krabičky, jak velká krabička, se má použít. Jinou krabičku potřebuješ na uložení telefonního čísla a jinou na text dopisu.
Pokud chceš s proměnnou pracovat, je třeba ji neprve založit - inicializovat. Pak jí můžeš přiřadit hodnotu, měnit hodnotu nebo jinak s ní pracovat. Inicializace vypadá následovně: 'typ jménoPromenne;' nebo 'typ jmenoPromenne = vychoziHodnota;'. Všechny primitivní datové typy mají výchozí hodnotu automaticky.
Lze změnit datový typ nějaké proměnné? Ano, přetypováním: '(novyTyp) jmenoPromenne'. Jakékoliv číslo lze přetypovat na jiný typ čísla. Cokoliv lze přetypovat na String.

//inicializace promennych
boolean anoNe = true;
int cislo1 = 33;
float cislo2= f1.0;
double cislo3 = -45.11;
long cislo4 = 54632300000000000000000000000000000000;
char znak = 'a'; // je nutne pouzit jednoduche uvozovky

// C#
string text = "zcela libovlny text"; // je nutne pouzit slozene uvozovky
// Java
String text = "zcela libovlny text"; // je nutne pouzit slozene uvozovky

// prace s promennou
int vekManzela = 30;
int vekZeny = 28;
int vekovyRozdil = vekManzela - vekZeny; // hodnota je 2

// ruzne zpusoby inicializace
// inicializovani = deklarovani promene
String name;
// prirezeni hodnoty
name = "some text";
// zmena hodnoty
name = "another text";
// deklarovani promene a prirazeni hodnoty
String forname = "second text";

String pozdrav = "Ahoj";
// jak na konec pridat vykricnik?
//prvni zpusob
pozdrav = pozdrav + "!";
//druhy zpusob
pozdrav += "!";

//pretypovani
int vychoziCislo = 42;
String cislo = (String) vychoziCislo;
double cislo1 = 12.2;
int cislo2 = (double) cislo1; // cislo2 je 12!

Úloha

Jakě typy jsou nejvýhodnější pro uložení telefonního čísla, ceny, emailu, poznávací značky, pohlaví?
Možná řešení:
  1. Telefonní číslo

    2. Integer, lze i String, pokud chceš psát číslo s mezerami nebo s předvolbou.

  2. Ceny

    3. Float, double, long

  3. Email

    4. String

  4. Poznávací značky

    5. String

  5. Pohlaví

    6. Boolean, stačí se rozhodnout, zda true znamená muž nebo žena :-). Lze použít i String, do kterého si uložíš například "muž", "žena". Pak můžeš mít i možnost "neuvedeno".

Napiš algoritmus, který prohodí hodnoty dvou proměnných 'a' a 'b' libovolného typu.
//zadani
String a = "text 1";
String b = "text 2";
//reseni
String pomocna = a;
a = b;
b = pomocna;
//nyni je v 'a' "text 2" a v 'b' "text 1"

Lekce 2: Datové struktury

Někdy je potřeba pracovat s více souvisejícími proměnnými. Lze definovat první proměnnou, druhou až třeba stou. Místo tohoto nešikovného řešení je lepší použít něco, co drží více proměnných naráz - datovou strukturu. Zde si ukážeme tzv. pole. V Javě a C# může být v poli jen jeden datový typ a má fixní velikost - od začátku má danou velikost (tebou zadanou) a tu nelze měnit. Pole obsahuje prvky. Ke každému se lze dostat zadáním pozice - indexu. POZOR: indexuje se od 0.

Vložení prvku do pole na danou pozici: 'jmenoPole[pozice] = hodnota'. Získání hodnoty na dané pozici: 'jmenoPole[pozice]'. Bývá užitečné vědět, jak je pole velké. Velikost pole získáš: 'jmenoPole.length'.

// inicializace pole integeru
int[] poleIntu;
int[] jinePole = new int[5]; //pole integeru o velikosti 5
//naplneni pole
jinePole[0] = 4;
jinePole[1] = 757;
jinePole[2] = -57;
jinePole[3] = 8888;
jinePole[4] = 0;
// ziskani druheho cisla
int druheCislo = jinePole[1];

//vytvoreni pole a naplneni prvky
double pole = new double[] {10, 12.01. -4};

Úloha

Pokud mám pole o 100 prvcích (length je 100), jaké číslo mám doplnit za x pokud chci poslední prvek pole: pole[x] ?
Zde je jediné správné: x = 99;
Vytvoř stringové pole o 3 prvcích a naplň jej slovy "ahoj", "zdar", "hoj"
//prvni moznost
String[] slova = new String[3];
slova[0] = "ahoj";
slova[1] = "zdar";
slova[2] = "hoj";

//druhy zpusob
String[] slova2 = new String[] {"ahoj", "zdar", "hoj"};

Lekce 3: Podmínky a větvení

Základní konstrukcí ve všech jazyků jsou podmínky - co když, co jinak. Ve většině jazyků včetně Javy a C# jsou dva způsoby If-else a switch.

If elseif else

Podmínka začíná if, v jednoduchých závorkách následuje výraz, který jazyk vyhodnocuje, zda je pravda nebo nepravda. Výrazem může být cokoliv, co lze vyhodnotit jako pravda/nepravda. Více proměných se skládá pomocí logického A - & - a logického NEBO - |. Opak se tvoří pomocí !.

// jednoducha podminka
if (vyraz) {
  // co se stane, pokud je podminka splnena
}

// podminka s else
if (vyraz) {
   // pokud je podminka splnena
} else {
   //jinak
}

// vice vetvena podminka
if (vyraz) {
   // pokud je podminka splnena
} else if (vyraz) { // muze se opakovat vicekrat
   // pokud prvni podminka neni splnena a druha je
} else {
   //jinak
}

Switch

Pokud máš nějakou proměnou a potřebuješ rozhodnout na základě její hodnoty, přičemž může nabívat mnoha hodnot, hodí se switch. Do závorek se doplní proměnná a jednotlivé hodnoty se uvádí v case části. Pokud je potřeba, lze uvést default, který se použije, pokud se žádná z předchozích hodnot neshoduje. Na konci kódu za case je nutné uvést braak, jinak se provede i kód u dalšího case.

switch (promenna) {
 case hodnota1: break; // zde se nic neprovede
 case hodnota2:
  // nejaky kod
  break;
 case hodnota4:
  // nejaky kod
  // neni break, provede se i kod u pristiho case
 case hodnota5:
  // zase nejaky kod
 default: // volitelne
  // kod
}

Výroková logika

Jméno Znak Popis
Logické A && Pravda, pokud obě části jsou pravda
Logické NEBO || Pravda, pokud alespoň jedna část je pravda

Lekce 4: Cykly

Každného trochu přemýšlivého člověka napadlo, že pokud má pole o např. 1 000 prvcích, je 'trochu' nepraktické naplňovat jej hodnotami po jednom (dlouhePole[0] = ..., dlouhePole[1] = ..., dlouhePole[1000] = ...). Právě na tom lze dobře pochopit funkci cyklů. Představíme si tři: for, while a foreach cyklus.

For cyklus

Nasledující kód vypíše pod sebe čísla od 0 do 9.
Uvnitř kulatých závorek - za slovem for - se nejprve definuje proměnná, pomocí které se prochází cyklus. Nejčastěji je to integer, ale může to být libovolný typ čísla. Za středníkem je podmínka. Dokud podmínka platí, cyklus bude probíhat. Za druhým středníkem se definuje, co se má stát s proměnnou. Do složených závorek se vkládá kód, který se má provádět.
Popis ukázky: for cyklus dosadí do proměnné promenna nulu, zkontruluje, zda je promenna menší než 10, vypíše do konzole nulu a proměnnou promenna zvětší o jedna. Tak to pokračuje, až je promenna = 10, v tu chvíli neplatí, že promenna < 10, a cyklus zkončí, aniž by vypsal do konzole desítku.

// for
for (int promenna = 0; promenna < 10; i = i+1) {
  System.out.println(promenna); // vypis v Javě, v C# je jine
}

While cyklus

Předchozí ukázka provede totéž jako ukázka for cyklu. While cyklus se provádí, dokud platí podmínka v závorkách. Rozdíl mezi for a while cyklem: for cyklus se provádí x krát, kde x je předem známé. U while cyklu není obvykle předem známo, kolikrát se provede. Tudíž ukázka while cyklu není úplně přesná.

// while
int promenna = 0; while (promenna < 10) {
  System.out.println(promenna);
  promenna++;
}

Foreach cyklus

Předhozí ukázka vypíše podsebe jednotlivé položky pole. Pomocí foreach cyklu je možné procházet cokoli, co je "iterable", což v našem tutoriálu je pouze pole. V kulaté závorce se nejprve definuje typ, který je uložen v poli, pak proměnná, do které se postupně ukládají jednotlivé položky pole. Za dvojtečku se uvede, které pole se má iterovat.

// for each
String[] pole = new String[] {"aa", "bb", "cc"};
for (String polozka : pole) {
  System.out.println(polozka);
}
