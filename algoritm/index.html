Lekce 1: Proměnné

V proměnné se udržuje hodnota, se kterou algoritmus pracuje. Tato hodnota je ale pro různé případy jiná - proměnná. Pro jednoduchost si proměnné představ jako krabičky do kterých se ukládají informace. Java a C# jsou tzv. typové jazyky, což znamená, že o každé proměnné musíš říci, jakého je typu. Při výuce algoritmů budeš nejčastěji používat primitivní datové typy, přičemž nejpoužívanější jsou:

  1. Boolean

    2. Nabývá hodnot 'true' a 'false'. Značí, zda něco je, či není.

  2. Integer (int)

    3. Celé číslo

  3. Float, double, long

    4. Desetinná čísla, nejčastěji používán double

  4. Character (char)

    5. Právě jeden znak (např. písmeno, číslice, bílý znak,...)

  5. String

    6. Textový řetězec, libovolné znaky v libovolném počtu. POZOR: v Javě string není primitivní datový typ.

Na všechna čísla lze použít základní matematické operace (+, -, /, *, modulo). Pomocí '+' se skládají stringy do nových stringů. Skládáním charů vznikají stringy.
Zůstaneme-li u přirovnání ke krabičkám, v podstatě říkáš, jaký druh krabičky, jak velká krabička, se má použít. Jinou krabičku potřebuješ na uložení telefonního čísla a jinou na text dopisu.
Pokud chceš s proměnnou pracovat, je třeba ji neprve založit - inicializovat. Pak jí můžeš přiřadit hodnotu, měnit hodnotu nebo jinak s ní pracovat. Inicializace vypadá následovně: 'typ jménoPromenne;' nebo 'typ jmenoPromenne = vychoziHodnota;'. Všechny primitivní datové typy mají výchozí hodnotu automaticky.
Lze změnit datový typ nějaké proměnné? Ano, přetypováním: '(novyTyp) jmenoPromenne'. Jakékoliv číslo lze přetypovat na jiný typ čísla. Cokoliv lze přetypovat na String.

//inicializace promennych
boolean anoNe = true;
int cislo1 = 33;
float cislo2= f1.0;
double cislo3 = -45.11;
long cislo4 = 54632300000000000000000000000000000000;
char znak = 'a'; // je nutne pouzit jednoduche uvozovky

// C#
string text = "zcela libovlny text"; // je nutne pouzit slozene uvozovky
// Java
String text = "zcela libovlny text"; // je nutne pouzit slozene uvozovky

// prace s promennou
int vekManzela = 30;
int vekZeny = 28;
int vekovyRozdil = vekManzela - vekZeny; // hodnota je 2

String pozdrav = "Ahoj";
// jak na konec pridat vykricnik?
//prvni zpusob
pozdrav = pozdrav + "!";
//druhy zpusob
pozdrav += "!";

//pretypovani
int vychoziCislo = 42;
String cislo = (String) vychoziCislo;
double cislo1 = 12.2;
int cislo2 = (double) cislo1; // cislo2 je 12!

Úloha

Jakě typy jsou nejvýhodnější pro uložení telefonního čísla, ceny, emailu, poznávací značky, pohlaví?
Možná řešení:
  1. Telefonní číslo

    2. Integer, lze i String, pokud chceš psát číslo s mezerami nebo s předvolbou.

  2. Ceny

    3. Float, double, long

  3. Email

    4. String

  4. Poznávací značky

    5. String

  5. Pohlaví

    6. Boolean, stačí se rozhodnout, zda true znamená muž nebo žena :-). Lze použít i String, do kterého si uložíš například "muž", "žena". Pak můžeš mít i možnost "neuvedeno".

Napiš algoritmus, který prohodí hodnoty dvou proměnných 'a' a 'b' libovolného typu.
//zadani
String a = "text 1";
String b = "text 2";
//reseni
String pomocna = a;
a = b;
b = pomocna;
//nyni je v 'a' "text 2" a v 'b' "text 1"

Lekce 2: Datové struktury

Někdy je potřeba pracovat s více souvisejícími proměnnými. Lze definovat první proměnnou, druhou až třeba stou. Místo tohoto nešikovného řešení je lepší použít něco, co drží více proměnných naráz - datovou strukturu. Zde si ukážeme tzv. pole. V Javě a C# může být v poli jen jeden datový typ a má fixní velikost - od začátku má danou velikost (tebou zadanou) a tu nelze měnit. Pole obsahuje prvky. Ke každému se lze dostat zadáním pozice - indexu. Pozor: indexuje se od 0.
Vložení prvku do pole na danou pozici: 'jmenoPole[pozice] = hodnota'. Získání hodnoty na dané pozici: 'jmenoPole[pozice]'. Bývá užitečné vědět, jak je pole velké. Velikost pole získáš: 'jmenoPole.length'.

// inicializace pole integeru
int[] poleIntu;
int[] jinePole = new int[5]; //pole integeru o velikosti 5
//naplneni pole
jinePole[0] = 4;
jinePole[1] = 757;
jinePole[2] = -57;
jinePole[3] = 8888;
jinePole[4] = 0;
// ziskani druheho cisla
int druheCislo = jinePole[1];

//vytvoreni pole a naplneni prvky
double pole = new double[] {10, 12.01. -4};

Úloha

Pokud mám pole o 100 prvcích (length je 100), jaké číslo mám doplnit za x pokud chci poslední prvek pole: pole[x] ?
Zde je jediné správné: x = 99;
Vytvoř stringové pole o 3 prvcích a naplň jej slovy "ahoj", "zdar", "hoj"
//prvni moznost
String[] slova = new String[3];
slova[0] = "ahoj";
slova[1] = "zdar";
slova[2] = "hoj";

//druhy zpusob
String[] slova2 = new String[] {"ahoj", "zdar", "hoj"};
