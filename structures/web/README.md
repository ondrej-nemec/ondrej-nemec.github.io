## How to install
### Download:

<a href="https://ondrej-nemec.github.io/download/--.jar" target=_blank>Download jar</a>

### Maven:
After `build` tag:
```xml
<repositories>
	<repository>
	    <id>jitpack.io</id>
	    <url>https://jitpack.io</url>
	</repository>
</repositories>
```
And to `dependencies`:
```xml
<dependency>
  <groupId>com.github.ondrej-nemec</groupId>
  <artifactId>--</artifactId>
  <version>--</version>
</dependency>
```
